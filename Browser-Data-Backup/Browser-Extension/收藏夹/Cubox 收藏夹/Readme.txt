概述
一键收藏到 cubox.pro，快速搜索你的收藏。
快速收藏你正在浏览的网页或链接，方便日后随时使用。你可以轻松将所需内容保存到 Cubox，并可在您使用的所有平台同步，包括网页、iPhone、iPad、Mac。您还可以通过书签描述记录详细的网站信息，并利用收藏夹、标签等方式，快速将书签分类，方便日后查找。你可以设置自己的分类规则，让 Cubox 每次为你自动分类所有收藏，无需繁琐地手动整理。

/ 快捷收藏 /

• 一键保存当前网页到 Cubox
• 自动根据网页内容，建议相关标签
• 自动填充网站标题和描述
• 自动抓取光标选中的文本，并填充为网站描述
• 手动更新已收藏网站的信息，包括标题、描述、位置和标签
• 通过鼠标右键，收藏当前网页中的任意链接
• 通过鼠标右键，收藏当前网页，并填充光标选中的文字
• 自动检测当前网页是否已收藏，可以随时更新信息，执行归档或删除等操作

/ 快捷搜索 /

• 在 Chrome 搜索框通过关键词 （C + 空格或TAB），直接在 Cubox 服务中搜索所有书签、智能分类、收藏夹、标签...
• 直接打开搜索结果中的某个书签，或定位到 Cubox 主页查看所有搜索结果

安装完成后，你需要先在 cubox.pro 的网页版登录你的 Cubox 账户，以授权 Cubox Chrome 插件使用你的账户。

你可以访问 cubox.pro 来登录网页版 Cubox，或下载 Cubox 的 iPhone, iPad, Mac 客户端。你的所有收藏会在不同设备上自动同步, 自动分类。

收藏网页
点击 Cubox 扩展按钮，或通过快捷键 Ctrl+Shift+X 收藏

收藏网页中内容
右键链接收藏链接；选中文字右键收藏页面并保存文字为描述

搜索书签
地址栏输入 c+空格 或 c+TAB 后在 Cubox 中搜索

管理书签
登录 cubox.pro 或 下载客户端，获得完善的阅读和管理功能
