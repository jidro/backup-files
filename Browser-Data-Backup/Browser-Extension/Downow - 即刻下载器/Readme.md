## Chrome插件安装

配合 Downow 使用，捕获 Chrome 浏览器的下载任务，转发链接到 Downow 下载器.    

#### 安装步骤

- 1.解压插件压缩包    
- 2.进入Chrome浏览器插件页面    
- 3.选择开启开发者模式    
- 4.加载解压的插件
