# uBlock Origin 广告拦截

## 简介：
一款高效的请求过滤工具：    
占用极低的内存和CPU，和其他常见的过滤工具相比，它能够加载并执行上千条过滤规则。    
用法：点击弹出窗口中的电源按钮，uBlock₀     
将对当前网页永久禁用/启用过滤功能。     
它只控制当前网页的请求过滤，而不是一个全局开关。
